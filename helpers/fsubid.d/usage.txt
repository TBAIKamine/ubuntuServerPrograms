fsubid - Generate next available subuid/subgid range

USAGE:
  fsubid [username]

ARGUMENTS:
  username    Target user (optional, defaults to SUDO_USER)
              Also fixes the first user's range to 100000:70000

OUTPUT:
  Prints a range like "170000:70000" (START:SIZE format)
  podmgr will convert this to START-END format for usermod

EXAMPLE:
  RANGE=$(fsubid dms)
  # podmgr handles the conversion to START-END format for usermod
